# Hospital Management System

A web-based Hospital Management System developed as part of the **Modern Application Development I (MAD1)** course offered by the **Indian Institute of Technology Madras (IIT Madras)**.

The application provides a platform for managing hospital operations, including patient records, doctor information, appointments, and administrative activities through an intuitive web interface.

## Features

* Patient Management

  * Add, update, view, and delete patient records
* Doctor Management

  * Maintain doctor information and details
* Appointment Management

  * Schedule and manage appointments
* User-Friendly Interface

  * Simple and responsive web interface
* Database Integration

  * Efficient storage and retrieval of data

## Technologies Used

* Python
* Flask
* HTML
* CSS
* JavaScript
* SQLite

## Installation and Setup

### Step 1: Create a Virtual Environment

```bash
python -m venv venv
```

### Step 2: Activate the Virtual Environment

```bash
venv\Scripts\activate
```

### Step 3: Install Dependencies

```bash
pip install -r requirements.txt
```

### Step 4: Run the Application

```bash
python app.py
```
